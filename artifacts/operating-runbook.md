# Operating Runbook\n\n## Weekly Rhythm\n\n1. Monday: Generate a new draft batch.\n2. Tuesday: Run review and quality gate checks.\n3. Wednesday: Approve and publish passing items.\n4. Thursday: Measure throughput, pass-rate, and cycle-time.\n5. Friday: Export handoff pack and queue next actions.\n\n## Escalation\n\n- If client approval is delayed beyond 48 hours, set item to request_changes.\n- Log the delay in the loop execution log.\n- Continue pipeline with next highest-priority item.\n\n## Async Handoff Contract\n\n- Handoff package must include board snapshot, metrics, and next actions.\n- No live meeting required to continue next loop.\n- Operator can resume work from handoff pack only.
